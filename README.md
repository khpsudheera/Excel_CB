# Excel_CB
Data analytics using Advanced Excel
1.Cleaning the Data: 
->Use Ctrl + F and replace missing data with "Not Available" when no data exists.
->Use =TRIM() to remove any extra spaces before names.
->Example: =TRIM(@Studio) — the column name should be placed inside the parentheses.
->Use Ctrl + Alt + V for Paste Special.
->Apply Conditional Formatting to highlight duplicate values and values that are less than or greater than a specified number.
->Remove duplicate values using the appropriate Excel option.
->Use Text to Columns → Delimited to split text into separate columns.
